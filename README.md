# 💰 Expense Tracker Web App

A simple **Expense Tracker application** built using **HTML, CSS, and JavaScript** that helps users track their income and expenses. The app calculates the **total balance, income, and expenses automatically** and stores transaction data using **Local Storage** so it remains saved even after refreshing the page.

---

## 🚀 Features

* ➕ Add new transactions (income or expense)
* 📊 Automatically calculates **total balance**
* 💵 Shows **total income and expenses separately**
* 🗑 Delete transactions
* 💾 Data stored in **Local Storage**
* 🔄 Transactions remain after page refresh
* 🎨 Clean and responsive UI

---

## 🛠 Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Application logic and DOM manipulation
* **Local Storage API** – Storing transaction data
